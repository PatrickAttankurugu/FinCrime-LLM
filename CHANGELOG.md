## [0.1.0] - 2024-11-06

### Added
- Project scaffolding and repository structure
- Mistral 7B fine-tuning scripts with QLoRA
- KYC risk assessment module
- Transaction analysis for suspicious patterns
- FastAPI backend with SAR, KYC, Transaction, and Compliance endpoints
- Streamlit interactive demo application
- Docker and docker-compose deployment support
- Training scripts with QLoRA (r=16, alpha=32)
- Synthetic data generation scripts (GPT-4 powered)
- Data preparation and validation pipelines
- Evaluation metrics (ROUGE, BLEU)
- GitHub Actions CI/CD workflows
- Support for 10 African countries
- Coverage for 10+ financial crime typologies
- Jupyter notebooks for exploration and tutorials
- Comprehensive test suite
