# Changelog

All notable changes to FinCrime-LLM will be documented in this file.

## [1.0.0] - 2024-11-06

### Added
- Initial release of FinCrime-LLM
- Mistral 7B fine-tuning for SAR generation
- KYC risk assessment capabilities
- Transaction analysis for suspicious patterns
- FastAPI production backend with 5 endpoints
- Streamlit interactive demo application
- Docker and docker-compose support
- Comprehensive training scripts with QLoRA
- Data generation and preparation scripts
- Evaluation metrics and baselines
- Full documentation suite
- GitHub Actions CI/CD workflows
- Support for 10 African countries
- 10+ financial crime typologies

### Features
- QLoRA training for efficient GPU usage
- WandB integration for experiment tracking
- Multi-task training capabilities
- Batch inference support
- Rate limiting and authentication
- CORS support for web applications
- Health checks and monitoring
- Comprehensive test suite

### Documentation
- Installation guide
- Training guide
- API documentation
- Dataset preparation guide
- Contributing guidelines
